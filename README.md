# Streamlit Stock Analysis App

This is a Streamlit web application for analyzing stock data using the `yfinance` library.

## Features

-   **Dividend & Price Analysis**:
    -   Displays historical Close and Adjusted Close prices.
    -   Visualizes dividend payments with estimated annualized yield.
    -   Calculates annual returns, volatility, and Sharpe ratios.
-   **Historical Valuation Analysis (P/E & P/S)**:
    -   Plots historical Price-to-Earnings (P/E) ratio against stock price and quarterly EPS.
    -   Plots historical Price-to-Sales (P/S) ratio against stock price and quarterly revenue.
-   **Configurable Tickers & Dates**:
    -   Allows separate input lists for Dividend and P/E/P/S analysis.
    -   Allows setting a custom start date for each analysis.
    -   Inputs (tickers and dates) are persisted across app restarts using a `config.json` file.
-   **Organized Output**: Results are displayed in a tabbed interface.
-   **Clear Visualizations**: Uses `matplotlib` for generating interactive charts with consistent styling.

## Installation

1.  **Clone this repository** (after it's pushed):
    ```bash
    git clone https://github.com/hillenr14/asset_research.git
    cd asset_research
    ```
    (Note: As we haven't pushed yet, you'll create the repository first, then you can follow these steps if you were a new user.)

2.  **Create a virtual environment** (recommended):
    ```bash
    python -m venv venv
    source venv/bin/activate   # On Windows: `venv\Scripts\activate`
    ```

3.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  **Run the Streamlit app**:
    ```bash
    streamlit run app.py
    ```

2.  The app will open in your web browser.

3.  In the sidebar, you can:
    -   Enter lists of stock tickers (one per line) for Dividend & Price Analysis.
    -   Enter lists of stock tickers (one per line) for Historical Valuation Analysis.
    -   Select a start date for each analysis.

4.  Click the respective "Analyze" button to generate the charts and tables.

5.  Your input tickers and dates will be saved automatically in `config.json` (which is excluded from Git) for persistence across app restarts.

## Project Structure

-   `app.py`: The main Streamlit application script.
-   `requirements.txt`: Lists all Python dependencies.
-   `config.json`: (Generated by the app) Stores user-specific configurations (excluded from Git).
-   `.gitignore`: Specifies files to be ignored by Git.
